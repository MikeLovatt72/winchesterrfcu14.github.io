## Adding downloadable documents

To make a PDF or Word doc available for download:

1. Upload the file to the repository on GitHub (**Add file → Upload files**)
2. In `index.html`, find the resource card you want to link (search for the document title)
3. Replace the `<span class="resource-tag">` line with:
   ```html
   <a href="your-filename.pdf" download class="resource-link">Download PDF →</a>
   ```
4. Commit changes — done
